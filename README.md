## Что это

**ПИКЕТ** — инструмент для машинистов скоростных поездов Октябрьской ж/д. Ведёт точный счёт километров и пикетов по GPS, заблаговременно предупреждает о введённых ограничениях скорости: голосом, звуком и визуально.

Веб-версия — та же программа, что Android APK, но запускается прямо в браузере. Установка не нужна. Работает на iPhone, Android, любом телефоне с браузером.

> ⚠️ **Помощник, не замена.** Приложение не заменяет АЛСН / КЛУБ-У и действующий регламент.

## GPS: как работает и как защищён

Приложение использует тот же спутниковый GPS-чип, что Яндекс.Навигатор — конкуренции нет.

**Многослойная защита от помех и глушения (проверена стресс-тестом 27/27 ✅):**
- Потолок 255 км/ч — выше физически невозможно для Сапсана (~250 конструктивный максимум)
- Фильтр скачков — отброс изменений >25 км/ч/с при нормальной езде (6× запас от экстренного торможения)
- Запасной канал — при молчании основного GPS >10с включается второй с пониженными требованиями
- Быстрое восстановление — после потери сигнала скорость восстанавливается за 1–2 замера, не через 8 секунд watchdog

## Когда нужен интернет

| | Нужен? |
|---|:---:|
| Счёт км/пк, сигнализация, голос | ❌ |
| GPS по спутникам | ❌ |
| **GPS через вышки сотовой сети** (резервный канал при потере спутника) | ✅ |
| Шрифты интерфейса (загрузка в первый раз, потом кэш) | ✅ |
| Первый запуск приложения | ✅ |

> **Про вышки:** при потере спутникового сигнала (туннель, глушение) телефон может определить приблизительную позицию через базовые станции сотовой сети — ПИКЕТ использует это автоматически. Точность 50–150м против 5–15м у спутника, но лучше, чем ничего.

## Установка на главный экран

### iPhone (Safari)
1. Открой ссылку в **Safari** → кнопка «Поделиться» → **«На экран "Домой"»**

### Android (Chrome)
1. Открой в Chrome → три точки → **«Установить приложение»**

---

## Перед поездкой

1. Разреши **геолокацию** при первом запросе
2. **iPhone:** коснись экрана хотя бы раз — это разблокирует звук (ограничение Apple)
3. Отключи **автоблокировку экрана** на время поездки (Настройки → Экран → Никогда)
4. **Откалибруйся** у столба: «Калибровка» → введи км → «Установить»
