# ПИКЕТ — веб-версия

Веб-версия помощника машиниста ПИКЕТ — открывается прямо в браузере телефона, без установки. Содержит ту же логику, что и Android-приложение: счёт километров/пикетов по GPS, переходы между участками, справочник скоростей, предупреждения об ограничениях (звук, голос, текст).

**Это помощник, а не замена АЛСН/КЛУБ и действующему регламенту.**

## Как открыть

Перейди по ссылке (вставится после публикации через GitHub Pages):

```
https://raiuchi.github.io/piket-web/
```

Сохрани её в закладки или на главный экран — при следующих открытиях сайт загрузится мгновенно и продолжит работать даже без интернета (вся карта станций уже встроена в страницу при первой загрузке).

---

## ⚠️ ВАЖНО: разница между веб-версией и Android-приложением

Веб-версия — это та же программа, но без отдельной фоновой службы, которая есть в Android-приложении (APK). Это значит:

- **Android-приложение (APK)**: работает с потухшим экраном, в фоне, при переключении на другие приложения. Можно убрать телефон в карман.
- **Веб-версия (любой телефон)**: работает только пока вкладка браузера открыта и видна на экране, как обычный навигатор на подставке. Если свернуть браузер или экран потухнет — счёт остановится.

Если тебе нужна именно фоновая работа — на Android поставь полноценное приложение: [github.com/Raiuchi/piket/releases](https://github.com/Raiuchi/piket/releases). Веб-версия — для тех, у кого iPhone, или кто хочет попробовать без установки.

## Инструкция для iPhone (Safari)

1. Открой ссылку выше в **Safari** (не в Chrome — на iPhone все браузеры используют движок Safari внутри, но прямой Safari надёжнее работает с разрешениями).
2. При первом запуске телефон спросит разрешение на **геолокацию** — обязательно разреши, иначе счёт работать не будет. Если случайно отказал — зайди в Настройки телефона → Safari → Местоположение → разреши заново.
3. Звук и голос на iPhone требуют, чтобы ты **сам хотя бы раз коснулся экрана** после открытия страницы (это ограничение Apple для всех сайтов, не наша недоработка) — просто нажми один раз на любую кнопку перед поездкой, например открой/закрой Настройки.
4. **Установи как приложение** (см. раздел ниже) — так удобнее открывать и выглядит привычнее.
5. **Во время поездки**: положи телефон на подставку экраном вверх, яркость не на минимум, отключи автоблокировку экрана на время поездки (Настройки → Экран и яркость → Автоблокировка → Никогда) — или просто будь готов касаться экрана, чтобы он не гас. Если экран погаснет или ты свернёшь Safari — отслеживание остановится, при возврате нажми Старт заново и при необходимости введи km/pk вручную (или дождись автоопределения).
6. **Голос**: на iPhone выбор русских голосов может быть скромнее, чем на Android — это особенность системы Apple, не программы.

## Как добавить ПИКЕТ на главный экран (как приложение)

Так сайт превращается в иконку на главном экране, открывается без адресной строки браузера — выглядит и запускается как обычное приложение.

**На iPhone:**
1. Открой ссылку в Safari.
2. Нажми кнопку **«Поделиться»** внизу экрана (квадрат со стрелкой вверх).
3. Прокрути вниз и выбери **«На экран “Домой”»**.
4. Нажми **«Добавить»** в правом верхнем углу.
5. На главном экране появится иконка ПИКЕТ — открывай её вместо Safari.

**На Android (Chrome):**
1. Открой ссылку в Chrome.
2. Нажми на три точки в правом верхнем углу.
3. Выбери **«Установить приложение»**, если он есть в списке (это даёт правильную иконку из манифеста). Пункт «Добавить на главный экран» тоже работает, но иногда показывает обычную серую заглушку с буквой вместо настоящей иконки — если увидишь такое, попробуй именно «Установить приложение».
4. Подтверди — иконка появится на главном экране.

**Важно понимать:** это делает запуск удобнее (иконка, без адресной строки, быстрее открывается), но **не убирает** главное ограничение — отслеживание всё равно останавливается, если экран потухнет или приложение свернуть. Это ограничение самих браузеров на телефонах, не способа запуска.

## Android (если открываешь через браузер, не APK)

Те же ограничения по фоновой работе, что и на iPhone — экран должен оставаться включённым. Открой ссылку в Chrome, разреши геолокацию при запросе. Как добавить на главный экран — см. раздел выше.

## Что одинаково на обоих телефонах

- Счёт километров/пикетов, переходы между участками, справочник скоростей, список ограничений — работает идентично Android-приложению.
- Калибровку (ввод km/пк/м) нужно ввести один раз перед стартом, дальше — либо вручную поправлять по факту проезжаемого столба, либо дать приложению определить позицию самому через GPS за несколько секунд.
- Если в приложении осталась позиция с прошлой, уже завершённой поездки — используй кнопку «Новая поездка» в окне калибровки перед новым маршрутом.
