## ✨ Что умеет

| | |
|---|---|
| 🛰 | **Точная позиция по GPS** — снэппинг к реальным координатам пути, без накопления ошибки одометра |
| 🚦 | **Сигнализация на подходе** — зелёный / жёлтый / красный, голос + звук |
| 📍 | **Калибровка по километровому столбу** — привязался один раз, дальше GPS ведёт счёт |
| 🔄 | **Автоопределение перегона** — сам понимает, где ты, по координатам |
| 📋 | **Список ограничений** — сортировка от ближайшего, справочник скоростей |
| 🌙 | **Ночной режим** — тёмный интерфейс, не слепит в кабине |
| 📴 | **Полностью офлайн** — после первого открытия не нужен интернет |
| 📱 | **Без установки** — открывается в браузере, работает на iPhone и Android |

## 📲 Как установить на главный экран (рекомендуется)

Открывается одной иконкой, без адресной строки, быстрее запускается.

### iPhone (Safari)
1. Открой **https://raiuchi.github.io/piket-web/** именно в **Safari**
2. Нажми кнопку **«Поделиться»** (квадрат со стрелкой вверх) внизу экрана
3. Прокрути вниз → **«На экран "Домой"»** → **«Добавить»**

### Android (Chrome)
1. Открой ссылку в **Chrome**
2. Три точки → **«Установить приложение»** (не «Добавить на главный экран»)

## 📖 Перед первой поездкой

1. **Геолокация** — при запросе обязательно разреши, иначе счёт не работает
2. **Звук** — на iPhone нужно хотя бы раз коснуться экрана после открытия (ограничение Apple)
3. **Экран** — держи включённым во время поездки. Лучше отключи автоблокировку: Настройки → Экран и яркость → Автоблокировка → Никогда
4. **Калибровка** — введи км/пк столба один раз перед стартом, или дай приложению определить позицию самому через несколько секунд после нажатия «Старт»
